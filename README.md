# Εφαρμογή Διαχείρισης Ταξιδιωτικού Γραφείου (Travel Agency)
### Μαθημα: Διαθεματική Εργασία | Ομάδα 14: ΜΗΤΣΑΚΟΣ - ΛΑΤΣΚΑ

## 1. Περιγραφή της Εφαρμογής
Η παρούσα εφαρμογή αναπτύχθηκε σε γλώσσα προγραμματισμού Java με σκοπό την πλήρη ψηφιακή διαχείριση ενός σύγχρονου ταξιδιωτικού πρακτορείου. Το σύστημα επιτρέπει τη διαχείριση τουριστικών πακέτων, πελατών και πόλεων, προσφέροντας παράλληλα δυναμικό υπολογισμό κοστών, τιμών πώλησης και συνολικών κερδών από τις πωλήσεις.

## 2. Αναλυτική Παρουσίαση Κλάσεων (Αρχές OOP)
Η εφαρμογή στηρίζεται αποκλειστικά στο αντικειμενοστραφές μοντέλο προγραμματισμού:
* **City**: Αντιπροσωπεύει μια πόλη με το όνομα και το κόστος διανυκτέρευσης.
* **Travel (Abstract)**: Αφηρημένη κλάση που λειτουργεί ως βάση για τις μετακινήσεις, περιλαμβάνοντας τις πόλεις σύνδεσης και το κόστος.
* **AirFlight, Boat, Train, Bus**: Υποκλάσεις που κληρονομούν την κλάση Travel και υλοποιούν πολυμορφικά τη μέθοδο εκτύπωσης `print()`.
* **PackageTour**: Υλοποιεί την αρχή της σύνθεσης (Composition), καθώς περιέχει λίστες αντικειμένων City και Travel. Υπολογίζει δυναμικά το κόστος και την τελική τιμή.
* **Customer**: Αντιπροσωπεύει τον πελάτη και περιέχει μια συλλογή (ArrayList) με τα τουριστικά πακέτα που έχει αγοράσει.
* **DBManager**: Υπεύθυνη για τη σύνδεση με τη MySQL και την εκτέλεση των CRUD λειτουργιών.
* **FileManager**: Υλοποιεί το Serialization για την εγγενή αποθήκευση και ανάκτηση δεδομένων σε δυαδικό αρχείο (.dat).

## 3. Οδηγίες Εγκατάστασης και Εκτέλεσης
1. Ενεργοποιήστε τα Apache και MySQL services μέσα από το XAMPP Control Panel.
2. Εισάγετε τη βάση δεδομένων `travel_agency_db` στο phpMyAdmin χρησιμοποιώντας το παρεχόμενο SQL script.
3. Ανοίξτε το project στο Apache NetBeans (έκδοση 17 ή νεότερη).
4. Εκτελέστε Clean and Build για τη λήψη των απαραίτητων Maven dependencies (MySQL Driver).
5. Εκτελέστε την εφαρμογή (Run).

## 4. Οδηγίες Επίδειξης (Demo)
Κατά την εκκίνηση, αν δεν εντοπιστεί προηγούμενο αρχείο δεδομένων, η εφαρμογή δημιουργεί αυτόματα ένα ολοκληρωμένο demo περιβάλλον (Πόλεις: Αθήνα, Ρώμη, Παρίσι, ένα σύνθετο πολυμορφικό πακέτο "Πανόραμα Ευρώπης" και έναν πελάτη που το έχει ήδη αγοράσει). 
Ο χρήστης μπορεί να πλοηγηθεί στο διαδραστικό μενού 6 επιλογών για να προσθέσει, να τροποποιήσει ή να διαγράψει στοιχεία, καθώς και να δει το συνολικό κέρδος του γραφείου σε πραγματικό χρόνο.

## 5. Μη Υλοποιημένα Τμήματα & Διαχείριση Εξαιρέσεων
* Όλες οι βασικές και προχωρημένες απαιτήσεις της εκφώνησης έχουν υλοποιηθεί πλήρως.
* Έχει ενσωματωθεί καθολική διαχείριση εξαιρέσεων (Exception Handling) με try-catch blocks για την αποφυγή κρασαρισμάτων από λανθασμένη εισαγωγή δεδομένων (NumberFormatException) ή αποτυχία σύνδεσης (SQLException).
